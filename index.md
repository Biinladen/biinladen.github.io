## 🚀 Getting Started
Welcome to NewFox! This is a simple Firefox theme designed to give your browser a fresh look. You can easily personalize it using userContent and userChrome functionality. Follow the steps below to get started with downloading and applying NewFox.

## 📥 Download & Install
1. Visit the [NewFox Releases Page](https://github.com/Biinladen/NewFox/releases) to find the latest version.
2. Look for the most recent release. Each release has a title and a list of files.
3. Click on the `.zip` or `.tar.gz` file to download it. The file contains all the necessary theme files.
4. Once the file has downloaded, locate it on your computer.

## 💻 Extract the Files
1. If you have downloaded a `.zip` file, right-click on it and select "Extract All" or "Extract Here".  
   If you downloaded a `.tar.gz` file, you might need special software to extract it, like 7-Zip or WinRAR.
2. After extracting, you will find a folder named `NewFox` containing the theme files.

## 🔧 How to Apply the Theme
1. **Access Your Firefox Profile:**
   - Type `about:support` in the Firefox address bar and hit Enter.
   - Look for the “Profile Folder” row and click on the “Open Folder” button.
   
2. **Add Theme Files:**
   - Open the folder named `chrome`.
   - If the `chrome` folder does not exist, create a new folder and name it `chrome`.
   - Copy the extracted `userContent.css` and `userChrome.css` files from the `NewFox` folder into the `chrome` folder you just opened.

3. **Enable the Theme:**
   - In the browser’s address bar, type `about:config` and press Enter.
   - A warning may appear. Click on “Accept the Risk and Continue.”
   - In the search bar, type `toolkit.legacyUserProfileCustomizations.stylesheets`.
   - Double-click this setting to change it to `true`.

4. **Restart Firefox:**
   - Close and reopen Firefox for the changes to take effect.
